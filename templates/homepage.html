About Project

The Main Aim Of The Project Is To Detect Whether The Person Is Wearing A Mask Or Not

To Predict Whether The Person Is Wearing A Mask Or Not. We Are Making Using Of Deep Learning.

Among Different Neural Networks We Make Use Of MobileNet As Our Primary Neural Network Model.

MobileNet is more reliable for low computing devices. So for detection we used MobileNet as the primary network.

For Developing Web Application We Made Use Of Flask as a main framework and front-end technologies like HTML,CSS,JS

The Trained Model is used for predicting the uploaded image or image which was live captured or through the live cam detection.
